# OverTheWire Bandit Write-ups

This repository contains my personal write-ups for the **OverTheWire Bandit** wargame.

Bandit is designed to teach basic Linux usage, shell commands, and fundamental security concepts through hands-on challenges.

## Goals
- Improve Linux and command-line skills
- Build a strong foundation for cybersecurity
- Practice problem-solving and security thinking
- Create a public security learning portfolio

## Structure
Each level has its own markdown file containing:
- The goal of the level
- The approach used to solve it
- Commands executed
- Explanation of the solution
- Key takeaways

## Disclaimer
Passwords and sensitive data are **intentionally omitted**.  
This repository is for educational purposes only.
